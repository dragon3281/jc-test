📦 文件列表
================================================================================
ppvip6_register/

├── README.md                详细使用说明
├── run.bat                  Windows启动脚本（双击运行）
├── run.sh                   Linux/Mac启动脚本

└── tokens.txt               Token输出文件（运行后自动生成）

================================================================================
🚀 快速开始（3步）
================================================================================

步骤1: 安装Python依赖
---------------------
打开命令行，运行：
    pip3 install curl-cffi PyExecJS

或者在Windows上运行：
    pip install curl-cffi PyExecJS

步骤2: 运行脚本
---------------------
Windows系统：
    双击 run.bat

Linux/Mac系统：
    chmod +x run.sh
    ./run.sh

或直接运行Python脚本：
    python3 ppvip6_自动化注册.py

步骤3: 查看结果
---------------------
- 控制台会显示实时进度
- tokens.txt 保存所有成功注册的Token

================================================================================
⚙️ 配置修改
================================================================================

如需修改配置，编辑 ppvip6_自动化注册.py 文件：

1. 修改注册数量（第37行）：
   REGISTER_COUNT = 100  # 改成你想要的数量

2. 设置代理（第25-29行）：
   PROXIES = None  # 不使用代理
   
   # 或者使用代理：
   PROXIES = {
       'http': 'http://127.0.0.1:10808',
       'https': 'http://127.0.0.1:10808'
   }

3. 修改默认密码（第38行）：
   DEFAULT_PASSWORD = "133adb"

================================================================================
🔑 重要说明
================================================================================

【加密逻辑】
ppvip系列网站使用特殊的加密规则：
- DES加密：使用原始rnd字符串的前8字节
- RSA加密：使用反转后的rnd字符串
脚本已正确实现这两种加密！

【必需文件】
- 加密逻辑.js 和 des加解密.js 必须与脚本在同一目录
- 不要修改或删除这两个文件
- 运行时会自动读取这两个文件

【运行要求】
- Python 3.7 或更高版本
- 安装 curl-cffi 和 PyExecJS 依赖包
- 能够访问 ppvip6.com（可能需要代理）

================================================================================
🔍 常见问题
================================================================================

Q1: 提示"ModuleNotFoundError"？
A1: 运行 pip3 install curl-cffi PyExecJS

Q2: Windows上无法运行？
A2: 确保安装了Python，并添加到了系统PATH

Q3: 注册失败率高？
A3: 检查网络连接，可能需要配置代理

Q4: 如何在Windows上查看tokens.txt？
A4: 用记事本打开 ppvip6_register 目录下的 tokens.txt

Q5: 可以修改注册数量吗？
A5: 可以，编辑脚本修改 REGISTER_COUNT 参数

================================================================================
💡 使用技巧
================================================================================

1. 批量注册：
   - 修改 REGISTER_COUNT 为需要的数量
   - 运行脚本后会自动批量注册

2. 后台运行（Linux/Mac）：
   nohup python3 ppvip6_自动化注册.py > output.log 2>&1 &

3. 查看日志：
   tail -f output.log

4. 停止运行：
   按 Ctrl+C 或关闭窗口
